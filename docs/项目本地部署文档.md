# 前端部署

前端项目地址：https://github.com/threesome-qiniu/qiniuyun-Develop-

`clone`到本地之后，执行包安装命令：

```shell
npm i
```
执行启动命令：
```shell
npm run dev
```
执行成功后打开网址：http://127.0.0.1:5173/index 即可访问。
